## 🚀 How to Deploy

### Option 1: GitHub Pages (Free)
1. Create a repository named `yourusername.github.io`
2. Upload `index.html` to the repo root
3. Enable GitHub Pages in Settings → Pages → Branch: `main`

### Option 2: Netlify (Free, Drag & Drop)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `index.html` file
3. Get a live URL instantly

### Option 3: Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import or drag the project folder
3. Deploy in one click

## ✏️ Customization Tips

- **Update email/phone:** Search for `[Email Address Available on Request]` in `index.html` and replace
- **Add a photo:** Inside `.hero-right`, add an `<img>` tag above the stat grid
- **Change accent color:** Edit `--gold: #b8933f` in the `:root` CSS variables block
- **Add sections:** Copy any existing `<section>` block and modify content
